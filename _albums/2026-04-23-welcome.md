This is a sample album post. Use the [composer](/album/new/) to build a new one in your browser — type title and body, drop in images, hit **Generate**, and you get a zip containing the markdown file and renamed images ready to drop into the repo.

Each post lives as one markdown file in `_albums/` with an `images:` list in the front-matter. The grid page iterates them newest-first.
